## 核实方法（按优先顺序）

1. **LetPub** (letpub.com.cn) — 中文界面，搜期刊名即显示最新 IF + 分区
2. **Clarivate JCR** (jcr.clarivate.com) — 官方来源，需机构订阅
3. **Researcher.Life** (researcher.life) — 免费查询单刊 IF
4. **期刊官网** — 多数期刊在 About 页标注 "Impact Factor: X"
